# 定跡管理

スマホでもPCでも使える、将棋定跡の管理用Webアプリです。

## できること

- 将棋盤を操作して定跡を作成
- 相手の手に対して自分の応手を登録
- 分岐、接続、メモを管理
- 局面を指定して定跡を検索
- 定跡ごとに `.joseki` ファイルとして書き出し/読み込み
- 複数定跡をZIPでまとめて書き出し
- 局面ごとの評価値欄に対応

## 使い方

`index.html` をブラウザで開くだけで使えます。サーバーやビルドは不要です。

## GitHub Pagesで公開する

1. GitHubで新しいリポジトリを作成します。
2. `index.html`, `styles.css`, `app.js`, `json-kifu-format-lib.js`, `README.md` をアップロードします。
3. リポジトリの `Settings` → `Pages` を開きます。
4. `Deploy from a branch` を選び、`main` / `/root` を指定します。
5. 表示されたURLからアプリを開けます。

## データについて

データはブラウザの `localStorage` に保存されます。別端末へ移したい場合は、`.joseki` ファイルとして書き出して読み込んでください。

## ファイル形式

`.joseki` はJSON形式です。中身は定跡データの配列で、各局面に `sfen`, `branches`, `eval` などを持ちます。
